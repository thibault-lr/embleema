## Getting started

### Configuration

Create a `.env`, copying the [.env.example](.env.example) file (it contains all the keys of `.env` **without sensitive
information**).

### Start the API in development mode

```shell
yarn dev
```

## Building

Build the application:

```shell
yarn build
```
